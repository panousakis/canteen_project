Οδηγίες εγκατάστασης σε XAMPP Ubuntu

1. Αντέγραψε τον φάκελο canteen_project μέσα στο:
   /opt/lampp/htdocs/

2. Άνοιξε το XAMPP:
   sudo /opt/lampp/manager-linux-x64.run

3. Εκκίνησε Apache και MySQL.

4. Άνοιξε το phpMyAdmin:
   http://localhost/phpmyadmin

5. Δημιούργησε βάση ή κάνε import το αρχείο database.sql.

6. Εκτέλεσε την εφαρμογή:
   http://localhost/canteen_project/

Αρχεία έργου:
- index.php
- products.php
- orders.php
- config.php
- style.css
- includes/header.php
- includes/footer.php
- database.sql
