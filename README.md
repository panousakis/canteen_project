# Σύστημα Διαχείρισης Κυλικείου

## Περιγραφή
Η εφαρμογή αποτελεί ένα σύστημα διαχείρισης κυλικείου που αναπτύχθηκε στο πλαίσιο του μαθήματος "Τεχνολογίες Διαδικτύου στην Ψηφιακή Βιομηχανία".

Σκοπός της εφαρμογής είναι η διαχείριση προϊόντων και παραγγελιών, καθώς και η παρουσίαση βασικών στατιστικών δεδομένων.

---

## Τεχνολογίες
- PHP
- MySQL (MariaDB)
- HTML
- CSS
- XAMPP (Apache Web Server)

---

## Λειτουργίες

### Διαχείριση Προϊόντων
- Προσθήκη νέων προϊόντων
- Προβολή λίστας προϊόντων
- Ταξινόμηση με βάση:
  - Τιμή
  - Απόθεμα
  - Ημερομηνία καταχώρισης

### Διαχείριση Παραγγελιών
- Καταχώριση νέων παραγγελιών
- Προβολή λίστας παραγγελιών
- Ταξινόμηση με βάση:
  - Όνομα πελάτη
  - Σύνολο
  - Ημερομηνία

### Στατιστικά
- Συνολικός αριθμός προϊόντων
- Συνολικός αριθμός παραγγελιών
- Συνολικά έσοδα

---

## Εγκατάσταση

1. Αντιγραφή φακέλου στο:
   /opt/lampp/htdocs/

2. Εκκίνηση XAMPP:
   sudo /opt/lampp/manager-linux-x64.run

3. Άνοιγμα phpMyAdmin:
   http://localhost/phpmyadmin

4. Δημιουργία βάσης δεδομένων:
   canteen_system

5. Import του αρχείου:
   database.sql

6. Εκτέλεση εφαρμογής:
   http://localhost/canteen_project/
